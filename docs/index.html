Safety Narrative Generator

This is a Shiny application for generating standardized safety narratives from clinical trial data.

Features

  • Data Import: Support for CSV and Excel files
  • Customizable Templates: Edit and save narrative templates
  • Batch Processing: Generate multiple narratives at once
  • Interactive Visualizations: Explore adverse events data
  • Summary Statistics: Calculate subject counts and exposure-adjusted rates

Deploy This App

This Shiny app requires a server to run. You have several options to deploy and use this app:

Option 1: Run Locally

# In R or RStudio:
install.packages(c("shiny", "ggplot2", "DT", "dplyr", "tidyr", "bslib", "devtools", "readxl"))
shiny::runGitHub("safety-narrative-generator", "YOUR_USERNAME")

Option 2: Deploy to shinyapps.io

Visit shinyapps.io to deploy this app online.

Option 3: Use Posit Connect

For enterprise deployment, consider using Posit Connect.
